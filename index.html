Kickerboard - Project description

Kickerboard is a web application to manage teams and results from table football matches. Users can log in and join one or multiple teams. Users can also submit new games. A match involves always two teams and their respective scores, which can range from 0 to 10 goals. The web application creates a leaderboard and dynamically displays the top 10 champion teams, calculated by their total number of wins and losses. A win gives 1 point, a loss deduces 1 point. It includes the following functionalities:

  • Creating teams: each user can create his own teams, of which he is the captain of
  • Joining teams: each user can join other teams. At the moment there is no limit and no confirmation needed
  • Submitting games: each user can submit games they played by selecting the winner, loser and the respective scores
  • Confirming games: after a game is submitted, it has to be confirmed by the captain of the loser team to be valid
  • Top ten: a leaderboard in the home view of the application shows the 10 best teams
  • Role based security: a secured access for admins is provided with an additional debug navigation menu item to manage the games, teams and memberships via the default grails scaffolded views
