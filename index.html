Kickerboard - Project description

Kickerboard is a web application to manage teams and results from table football matches. Users can log in and join one or multiple teams. Users can also submit new games. A match involves always two teams and their respective scores, which can range from 0 to 10 goals. The web application creates a leaderboard and dynamically displays the top 10 champion teams, calculated by their total number of wins. It includes the following functionalities
